## 使用步骤
---------

#### Step 1.添加依赖<br>
>>项目的 build.gradle 添加
```java
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  >>/app/build.gradle
  ```java
  	dependencies {
	        compile 'com.github.leifu1107:ViewpagerTransformer:V1.0'
	}
```
